# WorldBorder

A Bukkit plugin to set a border limit to worlds in Minecraft.

## Getting Started

Execute `mvn clean install` to compile the project.

The resulting jar file will be found in the /target folder.

### Prerequisites

* Maven
* Git
* Java JDK 1.8
* PaperSpigot 1.7.10
